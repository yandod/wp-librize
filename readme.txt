Shows shelf information from Librize(http://librize.com/). 
リブライズから最新の書籍情報を取得して表示します。

== Description ==

公開されているリブライズのAPIを利用して指定した本棚から最新の書籍情報を任意の件数表示します。使うには`[librize place=場所ID limit=件数]`のショートコードを投稿やページ、テキストウィジェットの中に記述します。

== Installation ==

Just install from dashboard and activate it.
ダッシュボードからインストールして有効化してください。設定などは不要です。


== Frequently Asked Questions ==

= 指定すべきIDなどはどのように決めれば良いですか？ =

リブライズ上で表示したい本棚を探し、URLからIDの部分を確認して利用してください。
